Hello everyone

This simple js webapp single page stores the user name. which you can see on the console

The following instructions on how to use it will be given below

  • You are to choose from the four different options: add, remove, display and quit
  • The "add" option allows the user to add a name to the roster
  • The "remove" option allows the user to remove a name from the roster. Note: only one name can be removed at a time.
  • The "display" option displays the user name on the console. We will update that very soon. We are so sorry for the inconvenience
  • The "quit" option close the roster webApp.

    • To access console on your pc

    • right click on your mouse
    • scroll down and click inspect
